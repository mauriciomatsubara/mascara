== Installation ==
* Download the plugin
* Upload the folder "woo-payment-discounts" to wp-content/plugins (or upload a zip through the WordPress admin)

== Frequently Asked Questions ==
= How to Setup : =
* Activate plugin and you will have 'Discount Per Payment' settings link under WooCommerce tab.
* Set amount and discount type against payment method list.
* That's it you are done!
